# مواصفات المشروع (Project Specifications) - snabbaLexinTS

## 1. التحليل (Analysis)
- **الغرض**: تحويل تطبيق القاموس من نمط الصفحات المتعددة (Multi-page) إلى نمط التطبيق أحادي الصفحة (SPA) باستخدام React لزيادة السرعة وسلاسة التنقل.
- **التحدي**: الحفاظ على وظائف البحث المتقدم، الإحصائيات، وميزات "كلمة اليوم" مع دمج شاشة عرض تفاصيل كاملة تدعم التنقل الذكي وميزات تفاعلية متقدمة.
- **الأداء**: استخدام التمرير اللانهائي (Infinite Scroll)، التخزين المؤقت (Memoization)، وتحميل الموارد الثقيلة (مثل الصوتيات) عند الطلب (Lazy Loading).

## 2. التوثيق (Documentation)
### المكونات الرئيسية (UI Components)
- `HomeView.tsx`: المكون الرئيسي للبحث والتصفح. يدعم التصفية، الفرز، وعرض النتائج بتمرير لانهائي.
- `DetailsView.tsx`: شاشة تفاصيل الكلمة. تحتوي على نظام تبويبات (Info/Interact) وتدعم التنقل السريع بين الكلمات.
- `PronunciationLab.tsx`: مختبر النطق المتقدم. يحتوي على:
    - متخيل صوتي (Audio Visualizer) يعرض الموجات الصوتية.
    - مسجل صوتي (Audio Recorder) للمقارنة.
    - نظام تقييم النطق (Speech Recognition Score).
- `MiniQuiz.tsx`: اختبار سريع للكلمة الحالية، يولد أسئلة ومشتتات ذكية بناءً على نوع الكلمة.

### الوحدات البرمجية (Modules)
- `src/details_modules/`:
    - `SmartLinkProcessor.ts`: معالجة النصوص لجعل الكلمات السويدية قابلة للنقر.
    - `MasteryManager.ts`: إدارة مستوى إتقان الكلمات وتتبع التكرار المتباعد.
    - `RelatedWordsManager.ts`: خوارزمية ذكية لاقتراح كلمات مركبة أو من نفس الجذر.
- `src/type-color-system.ts`: النظام المركزي لتحديد ألوان البطاقات والحدود بناءً على القواعد النحوية (أحمر للأفعال، أخضر للأسماء، إلخ).

## 3. التنفيذ (Implementation)
- **تم التنفيذ**:
    - **التنقل (Navigation)**: أزرار "السابق/التالي" في شاشة التفاصيل للتنقل السلس.
    - **الروابط الذكية (Smart Links)**: القدرة على النقر على أي كلمة داخل التعريف للانتقال إليها دون إعادة تحميل الصفحة.
    - **مختبر النطق (Pronunciation Lab)**: دمج كامل لميزات Web Audio API و Speech Recognition.
    - **التكامل (Integration)**: ربط جميع المكونات بـ `React Router` (محاكاة عبر State) وإدارة التاريخ (`history.pushState`) لدعم زر الرجوع في المتصفح.
    - **التصميم (Styling)**: الالتزام بنمط "Left Border" الملون والتصميم المظلم (Dark Mode) المتوافق مع شاشات OLED.
